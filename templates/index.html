Choose Your Device Type

Arduino

The Arduino Uno is an open-source development board used to create various interactive devices.
It is designed to be user-friendly and low-cost, making it perfect for beginners and professionals alike.
Arduino Uno comes with a set of digital and analog input/output pins that can be used to control
various devices like lights, motors, and sensors. It is also compatible with a wide range of shields
(add-ons) that extend its functionality and provide additional features.

ESP32 WROOM

The ESP32 is a development board that uses the ESP32 chip from Espressif Systems. This board is used
in Internet of Things (IoT) applications and small to medium-sized electronic projects. The ESP32 features
strong performance and a wide range of features, making it suitable for a variety of applications.

BladeRF

BladeRF is a radio system that can be used in many applications, such as electromagnetic spectrum
communication testing, and security research. BladeRF comes in different versions, each varying in capability
and supported frequency range. It is easy to use with open-source software like GNU Radio and SDR#.
